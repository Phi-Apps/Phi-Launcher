# **Phi Launcher**

Phi is a minimal Android launcher built for people who want a fast, clean home screen without giving up flexibility. No bloat, no forced structure just a launcher that works the way you do.

## Features

- **Spaces:** Create completely independent home screen layouts and switch between them instantly with a long press.
- **Pages:** Create multiple freeform and app widgets screens in a single space.
- **Radial Wheel:** A circular quick-launch menu always one gesture away.
- **Phi Widgets:** Quick Notes, To-Do list, and Music Controller accessible with a swipe.
- **Global Search:** Search apps, contacts, calendar events, and shortcuts from a single screen.
- **Custom Home Screen:** 10+ clock styles, search bar options, shortcuts, folders, and app groups.
- **Dev Panel:** Track LeetCode, Codeforces stats, and friends’ coding progress instantly.
- **Gestures:** Use 12 different gestures to launch multiple quick actions.
- **Smart App Drawer:** List/grid/tiles/pager view, alphabetical/minimal scroller, usage stats and quick actions.
- **Deep Customization:** icons colors, style, fonts, icon packs, alignment, shuffle mode, and more.
- **Focus & Privacy:** Hide apps, minimal mode, focus mode, launch delay, and screen time insights.
- **Fast Performance:** Zero ads or trackers, smooth animations, instant launches, lightweight size.
